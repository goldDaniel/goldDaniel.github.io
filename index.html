Shape Shooter

Source
Written in Java using the LibGDX Framework

For this project I implemented a spring-mass system for the warping background effect. I have also interpolate half of the springs, giving a similar effect to a full simulation while drastically reducing CPU usage.

I also integrated a 3rd-part bloom post-processing library that I had to modify to get working with my project. In the future I would like to write the bloom effect myself

One last part that I would like to mention is that the small enemies are an implementation of boids, an algorithm with emergent movement patterns. While this was not hard to implement, it was fun to tune and put into the game!

Bullet Hell

Source
Written in Java using the LibGDX Framework

During this project I learned how to manage many on screen objects, and had to use pooling to avoid memory allocation caused framedrops. The system I developed to fire bullets in this game was extended into the shape-shooter particle system.

Puzzle Platformer

Source
Written in Java using the LibGDX Framework

The key features of this project is a 2d camera that can interpolate based off various points on the map, and the use of colored keys to open doors.
There is also a debug view that lets you see triggers around the map, outlines of bounding boxes, position data, and collision data.

Blast Dem Bots

Source
Written in Java using the LibGDX Framework

A top down shooter written as my first experiment with tilemaps. This game features bots that will follow you around the map, with their paths being given from the A* pathfinding algorithm. This was also my first time working with 2d cameras dealing with worlds larger than the screen.
